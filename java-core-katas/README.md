# java-core-katas

Java exercises and katas focused on backend fundamentals and software quality.

## What you'll find here
- **OOP**: small domain models with validation and behavior (e.g., Money, BankAccount)
- **Collections**: practical use of List/Set/Map (e.g., WordCounter)
- **Testing**: JUnit 5 tests as the primary way to validate behavior (no CLI)

## How to run
```bash
mvn test
